== Description ==
A Single page creative WordPress theme - Radiance Lite packs the credentials of being clean, multipurpose and responsive business theme. It works exceptionally well for real estate, corporate, portfolio or photography site. The sketchboard framework lets you conveniently choose multiple color options for your site. While fully responsive liquid layout gives Radiance Lite an equally appealing appearance on mobile or tablets, this WordPress theme also supports multi-level navigation. What more can you ask for...It is free!!!

== Installation ==
1. Go to Appearance >> Add New
2. Search for Radiance Lite.
3. Click Install and finally activate the theme.
		Or
1. Download the latest version from https://wordpress.org/themes/radiance-lite/
2. Go to Appearance >> Add New >> Upload Theme
3. Select the downloaded theme zip file and click Install Now

== Frequently Asked Questions ==
= How to set Theme Front Page? =
1. Select "A static page" to show on from "Settings -> Reading -> Front page displays"
2. Landing Page Content
	Create 3 Pages and Select as landing page from "Appearance -> Customize -> Home Page Setting -> Landing Page Section"
3. To set the Navigation
	Go to "Appearance -> Menus"
	>>For First Section :- Create a custom link in Menus and put #section1 in URL and Navigation Label according to you for landing page.
	>>For Second Section :- Create a custom link in Menus and put #section2 in URL and Navigation Label according to you for landing page.
	>>For Third Section :- Create a custom link in Menus and put #section3 in URL and Navigation Label according to you for landing page.
	>>For Home Blog Section :- Create a custom link in Menus and put #front-posts-box in URL and Navigation Label according to you for landing page.
	
= Which class can I use for featured box icons? =
You can find the class name here http://fortawesome.github.io/Font-Awesome/icons/

= How to use theme in different languages? =
The Radiance Lite is ready for translation. Translation files are in /languages.
Read the following blog for more https://sketchthemes.com/blog/effective-steps-to-translate-your-wordpress-theme-using-qtranslate/
